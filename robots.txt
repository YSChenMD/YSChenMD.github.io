# their unfurler is allowed below — those bots obey robots.txt, and a blanket Disallow
# (even with a single image Allow) blocks the preview because they must read the page
# HTML first. Telegram / iMessage / Signal preview regardless of this file.
#
# For a hard "do not index this image" guarantee you would also serve og-image with an
# "X-Robots-Tag: noindex" response header (honored by Google AND facebookexternalhit).
# GitHub Pages cannot set headers; that needs a proxy/CDN (e.g. Cloudflare) in front, or
# a host with a _headers file (Cloudflare Pages / Netlify).
